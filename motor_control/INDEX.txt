📁 Motor Control Library - Quick Navigation
============================================

/inc/                          📋 Header Files (Public API)
  ├── motor_control.h         ⭐ Master header - include this
  ├── encoder.h               ⏱️  Encoder interface
  ├── PID_ctl.h               🎛️  PID controller interface
  ├── motor_driver.h          ⚡ L298N driver interface
  └── motor_ctl.h             🎮 Main motor controller

/src/                          ⚙️  Implementation (Source Code)
  ├── encoder.cpp
  ├── PID_ctl.cpp
  ├── motor_driver.cpp
  └── motor_ctl.cpp

/config/                       ⚙️  Configuration
  └── motor_config.h          🔧 Edit this for tuning!

/examples/                     📚 Usage Examples
  └── motor_example.cpp       💡 Complete working example

/docs/                         📖 Documentation
  ├── README.md               📖 Getting started
  ├── PID_TUNING_GUIDE.md     📊 How to tune PID
  └── IMPROVEMENTS_SUGGESTIONS.md  🚀 Future features

STRUCTURE.md                   📋 This directory layout

============================================

QUICK START:
-----------
1. Copy library to your STM32 project
2. Add inc/ path to compiler includes
3. Edit config/motor_config.h for your hardware
4. Include: #include "motor_control/inc/motor_control.h"
5. Follow examples/motor_example.cpp
